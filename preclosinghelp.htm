Search Previous Customer Information Help.

In order to precess closing, you have to have a previous order for retrieving your customer information, like account info., borrower/seller info., and property info.. Input or select one or two or three of the criteria to search your previous customer information. If you do not specify any one of the creteria, that will give you all the list of the previous orders.

Borrower's Last Name : Enter the first borrower's last name

Borrower's First Name : Enter the first borrower's first name.

State of Property : Select the state that the property is located.
